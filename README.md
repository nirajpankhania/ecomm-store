# ecomm-store
an ecommerce store lol

## server
express - web framework
cors  - for dealing with cross origin 
dotenv - loads secrets
bcrypt - hashing pwds
jsonwebtoken - creating and verifying JWTs
pg - postgresql client
nodemone - restarts server auto for development

## client
axios - making http requests to backend
react-router-dom - for client side routing
react - framework
